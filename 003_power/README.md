# Power
Use the 'host' connector on the power supply side. Pins 2 to 5 can be left unconnected to increase compatability.
## pinning

| Pin | Description |
| --- | ----------- |
| 1   | GND         |
| 2   | GND |
| 3   | GND |
| 4   | GND |
| 5   | GND |
| 6   | 1.2V |
| 7   | 1.8V |
| 8   | 2.5V |
| 9   | 3.3V |
| 10  | 5V   |
